Functioning

I implemented an additive synthesis component that enables users to define and add harmonics to a sine wave for music composition. It supports configuring an Attack, Decay, Sustain, and Release (ADSR) envelope, as well as vibrato rate and frequency. The main class, CAdditiveSynthesis, contains a CAdditiveSineWave object that generates harmonics and incorporates vibrato via a differential sinewave generator.

The score file demonstrates the capabilities of the AdditiveSynthesis component. The first tune features harmonics (up to 4 in this file), an ADSR profile, and a high vibrato effect, resulting in a trembling sound. The second tune removes the vibrato, while the third omits the ADSR envelope, giving the sound a more raw, aggressive tone. Finally, the removal of upper harmonics results in a plainer sound with reduced high frequencies.

Score Format

The instrument can be specified in the score file as follows. 

              
<?xml version="1.0" encoding="utf-8"?>
<score bpm="60" beatspermeasure="4">
  <instrument instrument="AdditiveSynthesis">
  ...
  <instrument />
<score />

Each note allows you to specify the amplitudes of the harmonics as space-separated floats, controlling their volume relative to the fundamental note. 

                
<note measure="1" beat="1.00" duration="0.5" amplitudes="1 0.8 0.6 0.4" note="C4"/>
<note measure="2" beat="1.00" duration="0.5" amplitudes="1 0.0 0.8 0.0 0.6 0.0 0.2" note="C4"/>
The Attack, Decay, Sustain, and Release envelope can be defined as space-separated floats to control how the sound evolves over time. The Attack defines how quickly the sound reaches full volume, Decay controls the drop in volume after the initial attack, Sustain determines the volume while the note is held, and Release sets the fade-out speed when the note ends. 
                
<note measure="1" beat="1.00" duration="0.5" ADSR=".3 .4 .5 .8" amplitudes="1 0.8 0.6 0.4" note="C4" />
Vibrato can also be applied to each note as a subtle variation in pitch, defined by a rate (speed of modulation) and a frequency (intensity of modulation) as space-separated floats. 
                
<note measure="1" beat="3.00" duration="0.5" vibrato="9.0 8.0" amplitudes="1 0.7 0.5 0.3" note="D5" ADSR=".3 .5 .5 .4"/>

Grading Elements

  • 10 - Sinusoid playback on demand from the sequencer ✅
  • 20 - Definition of all harmonics ✅
  • 30 - Envelope generation ✅
  • 35 - Polyphony ❌
  • 40 - Sound cross-fading ❌
  • 50 - Vibrato ✅
