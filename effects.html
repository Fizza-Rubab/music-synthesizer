Additive Synthesis Effects Module

Effects Module

Fizza Rubab
Michigan State University

Functioning

I have implemented three effect classes—NoiseGate, Chorus, and Compression—within a parent Effects class, which contains each specific effect as an attribute. The Effects class also includes an EffectSelector structure that stores the effect index (ranging from 0 to 2 for NoiseGate, Compression, and Chorus, and -1 for none), as well as the wet and dry signal levels. For each instrument, the EffectSelector is populated through the score file, setting up the effects and their parameters. After the instrument generates its audio, the samples are passed through each effect by calling the Process function, where the dry signal is processed with the effect and then weighted by the wet factor, allowing flexible control over the mix of wet and dry signals for each effect.

  1. Noise Gate: attenuates signals below a certain threshold to remove unwanted noise. The attenuation is down gradually in a linear fashion. Below is a sample effect applied on a happy birthday melody.
  2. Compression: reduces the dynamic range of the audio by applying gain reduction to signals above a set threshold. It uses parameters that are threshold level, the compression ratio, attack, and release time.
  3. Chorus: The Chorus effect modulates the audio signal with a delayed version of itself, creating a thicker sound by simulating multiple voices. Parameters are delay time, rate/speed of modulation, and range/depth of modulation.

Score Files and Format

An effect specification in a score file is done in two places. In the effects tag, a specific effect tag is created with the name of the effect and its parameters. This is a global definition. Then within an instrument, an effect tag is added to specify which effect to include along with its wet and dry contributions. Here is an example. 

  
  <?xml version="1.0" encoding="utf-8"?>
  <score bpm="60" beatspermeasure="4">
    <effects>
      <effect name="noisegate" threshold="0.5" />
      <effect name="chorus" samplerate="44100" delay=".01" range=".8" rate="20" />
      <effect name="compression" threshold="0.5" ratio="0.25" attack="0.5" release="0.5"/>
    </effects>
    <instrument instrument="AdditiveSynthesis">
      <effect which="noisegate" wet="1.0" dry="1.0" />
      <effect which="compression" wet="0.6" dry="0.4" />
      <effect which="chorus" wet="0.8" dry="1.0" />
      <note measure="1" beat="1.00" duration="0.5" amplitudes="1 0.8 0.6 0.4" note="C4" ADSR=".2 .3 .5 .6"/>
      ...
    </instrument>
    ...
  </score>
