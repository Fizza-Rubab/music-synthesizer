Melodies of Yesterday

This solo project is a compilation of melodies we have all heard and enjoyed in childhood. Across different languages, cultures, and regions, these nursery rhyme melodies remain the same, offering a shared experience that connects us all. As you listen, I hope it reminds you that your childhood wasn't that long ago.

I have implemented the Additive Synthesis and Effects modules, incorporating Compression, Chorus, and Noise Gate. Since I am not very familiar with music composition, I used letter notes from https://noobnotes.net/ and arranged the measures and durations with assistance from ChatGPT (GPT-4). This piece includes a variety of harmonics—mixture of even/odd harmonics—along with varying ADSR envelopes. The effects in this piece add texture: a strong chorus effect enhances depth, medium compression smooths out dynamics, and a subtle noise gate is added. Some sections feature a pronounced vibrato effect, giving it a nostalgic, guitar-like quality, while the intro and outro remain vibrato-free. The score file above details the harmonics, ADSR settings, and effects applied to each section.

Score Format

The actual score file used can be accessed from above. The format used for the score file is XML as shown below 


<?xml version="1.0" encoding="utf-8"?>
<score bpm="60" beatspermeasure="4">
  <effects>
    <effect name="noisegate" threshold="0.5" />
    <effect name="chorus" samplerate="44100" delay=".01" range=".8" rate="20" />
    <effect name="compression" threshold="0.5" ratio="0.25" attack="0.5" release="0.5"/>
  </effects>
  <instrument instrument="AdditiveSynthesis">
  <effect which="noisegate" wet="1.0" dry="1.0" />
    <note measure="1" beat="1.00" duration="0.5" amplitudes="1 0.8 0.6 0.4" note="C4" ADSR=".2 .3 .5 .6"/>
    <note measure="1" beat="1.50" duration="0.5" amplitudes="1 0.7 0.5 0.3" note="E4" vibrato="9.0 8.0" ADSR=".1 .2 .2 .6"/>
    ...
  </instrument>
  ...
</score>

As was the case in the synthie project, user specifies the instrument name in the instrument tag. In my case, there is only one instrument "AdditiveSynthesis". Then in the subsequent notes, user can specify properties of the additive synthesis generation. These include amplitudes of harmonics, ADSR profile values, vibrato rate and effect. In the effects tag, user can add effect subtag with its properties out of compression, chorus and noisegate. Within the instrument, effect tag can specify the wet and dry effect send can be controlled by given the name of the effect. More details can be found in the individual pages.
